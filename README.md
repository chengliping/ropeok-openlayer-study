### introduce
使用vue2，实现对openlayers的一些常用功能的组件化，以便在其他工程能快速使用或者用来参考

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn start
```

### Run local api server
```
yarn api
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

####  基础框架版本 ropeok 3.0.0 vue2.0
